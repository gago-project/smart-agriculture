## 一、背景与数据可行性结论

### 业务问题

用户问法示例：
- "最近30天全省预警处置情况怎么样？"
- "上周南通市的预警处置进度如何？"
- "4月份徐州市处理了多少条预警？"

预期回答模板：
```
{时间范围}{区域范围} 内共出现 {X} 条墒情预警信息，处置情况如下：
已处理 {X1} 条，待处理 {X2} 条，超时已处理 {X3} 条，超时待处理 {X4} 条。
```

无数据应答：
```
{时间范围}{区域范围} 内未查询到有效墒情预警信息，无对应处置数据。
```

**强制规则**：四种状态的名称和展示顺序固定，不得调整：
`已处理 → 待处理 → 超时已处理 → 超时待处理`

### 数据来源

- **主数据文件**：`1111.xlsx`（1,068 行，过滤 `warn_type='墒情预警'` 后 **1,056 条**有效）
- **行政区划辅助表**：`sys_region.xls`（乡镇级，9 位行政编码，含层级关系）

### pub_status 字段语义（已业务侧确认）

| pub_status | 含义 |
|---|---|
| 1 | 待处理 |
| 2 | 超时未处理（对外展示为「超时待处理」） |
| 3 | 已处理 |
| 4 | 超时处理（对外展示为「超时已处理」） |

状态由上游系统按「72 小时规则」写入，**本 Agent 直接读 `pub_status` 字段，不重新计算**。

### 数据质量确认

- `warn_type='墒情预警'` 过滤后，`pub_status` 只有 1/2/3/4，无脏值
- `address` 字段格式为「苏州市昆山市」，1,056 条零解析失败
- `warn_level` 只剩「涝渍」/ 「重旱」，无杂项

## 二、数据库建表

### 2.1 sys_region 行政区划表（**鼓励落库**）

**为什么落库**：
1. `1111.xlsx` 的 `region_code`（9 位，精确到乡镇）→ 通过 sys_region 可准确还原标准 city / county，比解析 `address` 字符串更可靠
2. 有 `parent_code` 层级关系，未来支持「按乡镇查处置情况」无需改表结构
3. 文件仅 230 KB，落库成本极低

**建表 SQL**（新文件 `infra/mysql/init/006_add_sys_region.sql`）：

```sql
-- sys_region: 行政区划表，精确到乡镇（9位编码）
-- 数据来源：sys_region.xls
CREATE TABLE IF NOT EXISTS sys_region (
  id           VARCHAR(36)  NOT NULL,
  region_code  VARCHAR(12)  NOT NULL  COMMENT '行政区划编码，6位=省/市/县，9位=乡镇',
  region_name  VARCHAR(64)  NOT NULL  COMMENT '区划名称',
  parent_code  VARCHAR(12)      NULL  COMMENT '上级区划编码',
  region_level TINYINT      NOT NULL  COMMENT '1省 2市 3县/区 4乡镇/街道',
  lon          VARCHAR(24)      NULL  COMMENT '经度',
  lat          VARCHAR(24)      NULL  COMMENT '纬度',
  created_at   DATETIME         NULL,
  updated_at   DATETIME         NULL,
  PRIMARY KEY (id),
  UNIQUE KEY uk_region_code (region_code),
  INDEX idx_parent_code (parent_code),
  INDEX idx_region_level (region_level)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='行政区划表';
```

> **注意**：`region_level` 值需在导入脚本中按编码位数推断：
> - 2 位 → 省（11、12…）
> - 6 位 → 市或县（32 开头为江苏）
> - 9 位 → 乡镇/街道
> 如果 sys_region.xls 中有独立的 level 字段，直接取；若没有，按位数推断。

**实现者注意**：读取 `sys_region.xls` 需要 `xlrd` 库（`pip install xlrd==1.2.0`，xlrd 2.x 不支持 `.xls`）。导入脚本见 §3.2。

```python
"""
导入 sys_region.xls 到 sys_region 表。

用法：
    PYTHONPATH=apps/agent python scripts/import/import_sys_region.py \
        --file /path/to/sys_region.xls

依赖：pip install xlrd==1.2.0
"""
import argparse, os, sys
import xlrd
import pymysql
import pymysql.cursors

# ── 配置 ──────────────────────────────────────────────────────────────────────
DB_CONFIG = {
    "host":   os.environ.get("MYSQL_HOST", "127.0.0.1"),
    "port":   int(os.environ.get("MYSQL_PORT", 3306)),
    "user":   os.environ.get("MYSQL_USER", "smart_agriculture"),
    "password": os.environ.get("MYSQL_PASSWORD", ""),
    "database": os.environ.get("MYSQL_DATABASE", "smart_agriculture"),
    "charset": "utf8mb4",
    "cursorclass": pymysql.cursors.DictCursor,
}

def infer_level(code: str) -> int:
    """按编码长度推断行政层级。"""
    length = len(str(code).strip())
    if length <= 2:  return 1  # 省
    if length <= 6:  return 2  # 市 or 县（由上级决定，此处保守取 2）
    return 4                   # 乡镇

def main():
    parser = argparse.ArgumentParser()
    parser.add_argument("--file", required=True)
    args = parser.parse_args()

    wb = xlrd.open_workbook(args.file)
    ws = wb.sheet_by_index(0)
    headers = [str(ws.cell_value(0, c)).strip() for c in range(ws.ncols)]
    print(f"Headers: {headers}")
    print(f"Total rows: {ws.nrows - 1}")

    # 字段映射（若 xls 实际字段名不同，在此处修改）
    # 预期字段：id, region_code, region_name, parent_code, lon, lat, create_time, update_time
    # 如有 level 字段直接使用，否则按位数推断
    FIELD_MAP = {
        "id":           ["id"],
        "region_code":  ["region_code", "code"],
        "region_name":  ["region_name", "name"],
        "parent_code":  ["parent_code", "parent"],
        "region_level": ["level", "region_level"],   # 可能不存在
        "lon":          ["lon", "coord_x"],
        "lat":          ["lat", "coord_y"],
        "created_at":   ["create_time", "created_at"],
        "updated_at":   ["update_time", "updated_at"],
    }

    def col(row_dict: dict, key: str):
        for alias in FIELD_MAP.get(key, [key]):
            if alias in row_dict:
                val = row_dict[alias]
                return None if val == "" else val
        return None

    conn = pymysql.connect(**DB_CONFIG)
    inserted = skipped = 0
    try:
        with conn.cursor() as cur:
            for row_idx in range(1, ws.nrows):
                row_vals = [ws.cell_value(row_idx, c) for c in range(ws.ncols)]
                row = dict(zip(headers, row_vals))

                rid        = col(row, "id")
                rcode      = str(col(row, "region_code") or "").strip()
                rname      = str(col(row, "region_name") or "").strip()
                parent     = col(row, "parent_code")
                rlevel_raw = col(row, "region_level")
                rlevel     = int(rlevel_raw) if rlevel_raw else infer_level(rcode)
                rlon       = col(row, "lon")
                rlat       = col(row, "lat")
                created    = col(row, "created_at")
                updated    = col(row, "updated_at")

                if not rcode or not rname:
                    skipped += 1
                    continue

                cur.execute(
                    """
                    INSERT INTO sys_region
                      (id, region_code, region_name, parent_code, region_level,
                       lon, lat, created_at, updated_at)
                    VALUES (%s,%s,%s,%s,%s,%s,%s,%s,%s)
                    ON DUPLICATE KEY UPDATE
                      region_name=VALUES(region_name),
                      parent_code=VALUES(parent_code),
                      region_level=VALUES(region_level),
                      updated_at=VALUES(updated_at)
                    """,
                    (rid, rcode, rname, parent, rlevel, rlon, rlat, created, updated),
                )
                inserted += 1

        conn.commit()
        print(f"Done: inserted/updated={inserted}, skipped={skipped}")
    finally:
        conn.close()

if __name__ == "__main__":
    main()
```

### 3.2 导入 warning_disposal_record（1111.xlsx）

```python
"""
导入 1111.xlsx 到 warning_disposal_record 表。
仅导入 warn_type = '墒情预警' 的行。
city / county 优先从 address 字段解析（「苏州市昆山市」→ city=苏州市，county=昆山市）；
若解析失败，从 sys_region 表通过 region_code 查询。

用法：
    PYTHONPATH=apps/agent python scripts/import/import_warning_disposal.py \
        --file /path/to/1111.xlsx
"""
import argparse, os, re
from datetime import datetime
import openpyxl
import pymysql
import pymysql.cursors

DB_CONFIG = {
    "host":     os.environ.get("MYSQL_HOST", "127.0.0.1"),
    "port":     int(os.environ.get("MYSQL_PORT", 3306)),
    "user":     os.environ.get("MYSQL_USER", "smart_agriculture"),
    "password": os.environ.get("MYSQL_PASSWORD", ""),
    "database": os.environ.get("MYSQL_DATABASE", "smart_agriculture"),
    "charset":  "utf8mb4",
    "cursorclass": pymysql.cursors.DictCursor,
}

_JIANGSU_CITIES = [
    "南京市","无锡市","常州市","苏州市","镇江市","南通市",
    "扬州市","泰州市","徐州市","连云港市","淮安市","盐城市","宿迁市",
]

def parse_address(address: str) -> tuple[str | None, str | None]:
    """从「苏州市昆山市」格式解析 city / county。"""
    addr = str(address or "").strip()
    city = next((c for c in _JIANGSU_CITIES if c in addr), None)
    county = addr.replace(city, "").strip() if city else None
    return city, county or None

def lookup_city_county_from_region(cur, region_code: str) -> tuple[str | None, str | None]:
    """
    通过 sys_region 层级查询 city 和 county。
    region_code 是乡镇级（9位），向上找父级：
      乡镇 → 查 parent_code → 区县（level=3）→ 再查 parent_code → 市（level=2）
    """
    cur.execute(
        "SELECT region_code, region_name, parent_code, region_level "
        "FROM sys_region WHERE region_code = %s",
        (region_code,),
    )
    row = cur.fetchone()
    if not row:
        return None, None

    county = city = None
    # 向上最多走 2 层
    for _ in range(2):
        level = int(row.get("region_level") or 0)
        if level == 3:
            county = row["region_name"]
        elif level == 2:
            city = row["region_name"]
            break
        parent = row.get("parent_code")
        if not parent:
            break
        cur.execute(
            "SELECT region_code, region_name, parent_code, region_level "
            "FROM sys_region WHERE region_code = %s",
            (parent,),
        )
        row = cur.fetchone()
        if not row:
            break

    return city, county

def main():
    parser = argparse.ArgumentParser()
    parser.add_argument("--file", required=True)
    parser.add_argument("--truncate", action="store_true",
                        help="清空表后重新导入（幂等用）")
    args = parser.parse_args()

    wb = openpyxl.load_workbook(args.file, read_only=True, data_only=True)
    ws = wb["Sheet1"]
    all_rows = list(ws.iter_rows(values_only=True))
    headers = all_rows[0]
    data_rows = [dict(zip(headers, r)) for r in all_rows[1:]]
    soil_rows = [r for r in data_rows if r.get("warn_type") == "墒情预警"]
    print(f"Total rows: {len(data_rows)}, 墒情预警: {len(soil_rows)}")
    wb.close()

    conn = pymysql.connect(**DB_CONFIG)
    inserted = skipped = fallback_count = 0

    try:
        with conn.cursor() as cur:
            if args.truncate:
                cur.execute("TRUNCATE TABLE warning_disposal_record")
                print("Table truncated.")

            for row in soil_rows:
                rid = str(row.get("id") or "").strip()
                if not rid:
                    skipped += 1
                    continue

                sn        = str(row.get("sn") or "").strip().upper()
                warn_time = row.get("warn_time")
                create_t  = row.get("create_time")
                pub_time  = row.get("pub_time")
                pub_status = int(row.get("pub_status") or 0)
                warn_level = str(row.get("warn_level") or "").strip()
                warn_level_code = row.get("warn_level_code")
                warn_value = str(row.get("warn_value") or "").strip() or None
                region_code = str(row.get("region_code") or "").strip() or None
                address = str(row.get("address") or "").strip() or None
                do_advice = row.get("do_advice")
                pub_user = str(row.get("pub_user") or "").strip() or None
                pub_user_name = str(row.get("pub_user_name") or "").strip() or None
                pub_sms_flag = row.get("pub_short_message_flag")
                content = row.get("content")

                # city / county：先解析 address，失败则查 sys_region
                city, county = parse_address(address)
                if not city and region_code:
                    city, county = lookup_city_county_from_region(cur, region_code)
                    if city:
                        fallback_count += 1

                cur.execute(
                    """
                    INSERT INTO warning_disposal_record
                      (id, sn, warn_time, create_time, pub_time, pub_status,
                       warn_level, warn_level_code, warn_value,
                       city, county, region_code, address,
                       do_advice, pub_user, pub_user_name,
                       pub_short_message_flag, content)
                    VALUES
                      (%s,%s,%s,%s,%s,%s,%s,%s,%s,%s,%s,%s,%s,%s,%s,%s,%s,%s)
                    ON DUPLICATE KEY UPDATE
                      pub_time=VALUES(pub_time),
                      pub_status=VALUES(pub_status),
                      do_advice=VALUES(do_advice)
                    """,
                    (rid, sn, warn_time, create_t, pub_time, pub_status,
                     warn_level, warn_level_code, warn_value,
                     city, county, region_code, address,
                     do_advice, pub_user, pub_user_name,
                     pub_sms_flag, content),
                )
                inserted += 1

        conn.commit()
        print(f"Done: inserted/updated={inserted}, skipped={skipped}, "
              f"sys_region fallback used={fallback_count}")
    finally:
        conn.close()

if __name__ == "__main__":
    main()
```

```python
def build_warning_disposal_audit_sql(
    city: str | None = None,
```

### 4.2 同步执行

```python
def query_warning_disposal_stats(
    self,
    *,
    city: str | None = None,
    county: str | None = None,
    start_time: str | None = None,
    end_time: str | None = None,
) -> dict:
    """
    返回预警处置统计：
    {
        "total": 100,
        "已处理": 80,
        "待处理": 5,
        "超时已处理": 12,
        "超时待处理": 3,
    }
    表不存在时返回 total=0 的空统计。
    """
    clauses: list[str] = ["pub_status IN (1,2,3,4)"]
    params: list = []
    for col, val in [("city", city), ("county", county)]:
        if val:
            clauses.append(f"{col} = %s")
            params.append(val)
    if start_time:
        clauses.append("warn_time >= %s")
        params.append(start_time)
    if end_time:
        clauses.append("warn_time <= %s")
        params.append(end_time)
    where = "WHERE " + " AND ".join(clauses)
    sql = f"""
    SELECT
        COUNT(*)          AS total,
        SUM(pub_status=3) AS status_done,
        SUM(pub_status=1) AS status_pending,
        SUM(pub_status=4) AS status_overtime_done,
        SUM(pub_status=2) AS status_overtime_pending
    FROM warning_disposal_record
    {where}
    """
    _EMPTY = {
        "total": 0, "已处理": 0, "待处理": 0,
        "超时已处理": 0, "超时待处理": 0,
    }
    connection = self._connect()
    try:
        with connection.cursor() as cursor:
            cursor.execute(sql, tuple(params))
            row = cursor.fetchone()
        if not row:
            return _EMPTY
        return {
            "total":      int(row.get("total") or 0),
            "已处理":     int(row.get("status_done") or 0),
            "待处理":     int(row.get("status_pending") or 0),
            "超时已处理": int(row.get("status_overtime_done") or 0),
            "超时待处理": int(row.get("status_overtime_pending") or 0),
        }
    except Exception as exc:
        msg = str(exc)
        if "warning_disposal_record" in msg and ("1146" in msg or "doesn't exist" in msg):
            return _EMPTY
        raise DatabaseQueryError(f"MySQL 查询预警处置统计失败：{exc}") from exc
    finally:
        connection.close()
```

```python
@staticmethod
def _is_warning_disposal_query(text: str) -> bool:
    """
    预警处置查询：用户关心的是预警是否被处理/处置进度，而非预警事件本身。
    必须含处置主题词，或含预警词 + 处置意图词。
    """
    has_disposal_subject = any(t in text for t in _WARNING_DISPOSAL_SUBJECT_TOKENS)
    has_warning_word     = any(t in text for t in _WARNING_RECORD_QUERY_TOKENS)
    has_disposal_intent  = any(t in text for t in _WARNING_DISPOSAL_INTENT_TOKENS)
    has_rule_word        = any(t in text for t in _WARNING_RULE_QUERY_TOKENS)
    if has_rule_word:
        return False
    return has_disposal_subject or (has_warning_word and has_disposal_intent)
```

```python
async def _reply_warning_disposal(
    self,
    *,
    message: str,
    route_decision: Any,
    session_id: str,
    turn_id: int,
    current_context: dict[str, Any],
) -> dict[str, Any]:
    try:
        resolved_args, time_window, resolved_entities, resolution_meta = await self._resolve_filters(
            message=message,
            tool_name="query_soil_summary",
            current_context=current_context,
            turn_id=turn_id,
        )
    except ValueError as exc:
        return await self._build_filter_clarification_response(
            message=message,
            turn_id=turn_id,
            current_context=current_context,
            capability="warning_disposal",
            grain="aggregate",
            clarify_text=str(exc),
        )

    time_window = {
        "start_time": time_window.get("start_time") or (route_decision.extra or {}).get("time_start"),
        "end_time":   time_window.get("end_time")   or (route_decision.extra or {}).get("time_end"),
        "source":     time_window.get("source") or "explicit",
    }

    city   = resolved_args.get("city")
    county = resolved_args.get("county")

    stats = await self.repository.query_warning_disposal_stats_async(
        city=city,
        county=county,
        start_time=time_window.get("start_time"),
        end_time=time_window.get("end_time"),
    )

    scope_label = self._entity_label(resolved_entities) or "全省"
    range_label = self._scoped_range_label(scope_label, time_window)
    total = stats.get("total", 0)

    # ── 强制规则：四种状态的名称和顺序固定，不得修改 ──────────────────────────
    _STATUS_ORDER = ["已处理", "待处理", "超时已处理", "超时待处理"]

    if total == 0:
        final_text = (
            f"{range_label}未查询到有效墒情预警信息，无对应处置数据。"
        )
    else:
        status_lines = "，".join(
            f"{name} {stats.get(name, 0)} 条"
            for name in _STATUS_ORDER
        )
        final_text = (
            f"{range_label}共出现 {total} 条墒情预警信息，处置情况如下：\n"
            f"{status_lines}。"
        )

    block_id = f"block_warning_disposal_{turn_id}"
    block = {
        "block_id":   block_id,
        "block_type": "warning_disposal_card",
        "display_mode": "evidence_only",
        "total":      total,
        "stats":      {name: stats.get(name, 0) for name in _STATUS_ORDER},
        "time_window": time_window,
        "region": {"city": city, "county": county},
    }

    audit_sql = self.repository.build_warning_disposal_audit_sql(
        city=city,
        county=county,
        start_time=time_window.get("start_time"),
        end_time=time_window.get("end_time"),
    )

    base_context = {
        "topic_family":       "data",
        "active_topic_turn_id": turn_id,
        "primary_block_id":   block_id,
        "primary_query_spec": {},
        "time_window":        time_window,
        "resolved_entities":  resolved_entities,
        "derived_sets":       {},
        "compare_winner_entity": None,
        "closed":             False,
    }
    query_spec = {
        "spec_id":  f"qs_{turn_id}_warning_disposal",
        "dataset":  "warning_disposal_record",
        "capability": "warning_disposal",
        "grain":    "aggregate",
        "time_window": time_window,
        "entities": {"city": [city] if city else [], "county": [county] if county else [], "sn": []},
        "filters":  {},
        "sort":     {},
        "page":     {"page": 1, "page_size": 1},
        "provenance": {"source_turn_id": turn_id, "follow_up_mode": "standalone"},
    }
    query_state = self._build_query_state(
        turn_id=turn_id,
        capability="warning_disposal",
        grain="aggregate",
        slots=resolution_meta["slots"],
        time_window=time_window,
        slot_confidence=resolution_meta["slot_confidence"],
        slot_source=resolution_meta["slot_source"],
        query_profile=self._resolve_query_profile(
            message=message,
            route_metadata=self._query_profile_route_metadata(
                route="warning_disposal", action="stats", list_target=None,
            ),
            current_context=current_context,
            slots=resolution_meta["slots"],
            time_window=time_window,
            follow_up_mode=self._query_profile_follow_up_mode(
                route_metadata=self._query_profile_route_metadata(
                    route="warning_disposal", action="stats", list_target=None,
                ),
                operation=resolution_meta["operation"],
            ),
        ),
    )
    turn_context = self._finalize_context(
        base_context=base_context,
        current_context=current_context,
        turn_id=turn_id,
        query_state=query_state,
        parent_target_key=resolution_meta["parent_target_key"],
        result_refs=self._build_result_refs(turn_id=turn_id, block=block),
    )
    return {
        "turn_id":      turn_id,
        "answer_kind":  "business",
        "capability":   "warning_disposal",
        "final_text":   final_text,
        "blocks":       [block],
        "topic":        self._topic_payload(turn_context),
        "turn_context": turn_context,
        "query_ref":    {"has_query": True, "snapshot_ids": []},
        "conversation_closed": False,
        "session_reset":       False,
        "query_log_entries": [
            self._build_query_log_entry(
                session_id=session_id,
                turn_id=turn_id,
                query_index=1,
                query_type="warning_disposal",
                query_spec=query_spec,
                executed_sql_text=audit_sql,
                row_count=1,
                snapshot_id=None,
                time_window=time_window,
                filters={"city": city, "county": county},
                executed_result=stats,
                result_digest={"total": total},
            )
        ],
    }
```
