# AI 本地部署材料目录

这组材料用于回答两个核心问题：

1. 省厅是否适合建设 AI 本地部署体系
2. 农业大模型与农业智能体应该如何部署、如何防控、如何保证数据安全与数据真实性

## 当前推荐口径

当前最推荐的汇报结论是：

- 可以给省厅出 AI 本地部署方案
- 现阶段更适合建设“国产通用大模型本地部署 + 农业智能体 + 真实业务数据与规则驱动”体系
- 不建议一开始就直接投入高成本自研农业基础大模型
